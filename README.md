# 🌸 花音故事 — 手机浏览器版

## 立即体验（零安装）

### 方式 A：本机打开（推荐）

电脑或手机上打开命令行，进入此目录后运行：

```bash
# Python 方式（电脑已装）
python -m http.server 8080

# Node 方式
npx serve .
```

然后用手机浏览器访问 `http://你的电脑IP:8080`（确保手机和电脑在同一 WiFi）。

### 方式 B：部署到互联网

把整个目录上传到任意静态托管服务：
- **GitHub Pages**：推送到 `gh-pages` 分支
- **Vercel / Netlify**：拖拽文件夹上传
- **Cloudflare Pages**：连接 GitHub 仓库

### 安装到手机桌面（PWA）

在手机 Chrome 中打开网址后：
1. 点击右上角 `⋮`
2. 选择 **「添加到主屏幕」**
3. 桌面会出现「花音故事」图标，点击即可全屏使用
